# PRO-SERWIS - Design System

## Kolorystyka
- **Kolor główny (primary)**: #1e3a8a (głęboki niebieski)
- **Kolor akcentowy (accent)**: #ea580c (pomarańczowy)
- **Kolor tła (background)**: #f8fafc (jasny szary)
- **Kolor tekstu (text)**: #1f2937 (ciemny szary)
- **Kolor tekstu jaśniejszy**: #6b7280 (średni szary)
- **Kolor biały**: #ffffff

## Typografia
- **Czcionka główna**: 'Inter', sans-serif
- **Czcionka nagłówków**: 'Inter', sans-serif (bold)
- **Rozmiar nagłówka H1**: 3rem (48px)
- **Rozmiar nagłówka H2**: 2.25rem (36px)
- **Rozmiar nagłówka H3**: 1.875rem (30px)
- **Rozmiar tekstu podstawowego**: 1rem (16px)
- **Rozmiar tekstu małego**: 0.875rem (14px)

## Efekty i animacje
- **Transition**: 0.3s ease-in-out
- **Box shadow**: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
- **Border radius**: 0.5rem (8px)
- **Hover scale**: 1.05

## Przyciski
- **Primary button**: tło #ea580c, tekst #ffffff
- **Secondary button**: tło #1e3a8a, tekst #ffffff
- **Hover effects**: zmiana jasności o 10%

## Layout
- **Max width**: 1200px
- **Padding section**: 4rem 1rem
- **Gap między elementami**: 2rem
- **Grid columns**: 12
